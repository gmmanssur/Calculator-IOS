**Calculadora**

Projeto de criação de uma calculadora, inspirada no layout da calculadora do IOS, neste projeto foi utilizado as seguintes linguagens:

Linguagem de marcação: HTML5
Estilização: CSS3
Linguagem de programação: JavaScript

Esta calculadora foi criada com o intuito de:
  - Aplicar na prática os conceitos adquiridos durante os estudos;
  - Incentivar a fazer pesquisas, estudar quando houver qualquer impedimento;
  - Foi criado do zero e não apenas cópia;
  - Desafiar meus conhecimentos;

Quaisquer tipos de conselhos, dicas e críticas construtivas são bem-vindo.
